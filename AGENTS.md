# AGENTS

## 项目定位

`Simple Sit` 是一个面向 Fabric 1.21.11 的极简坐下模组，目标是提供稳定、简单、服务端优先的坐下体验。

## 协作原则

- 默认保持服务端优先设计
- 不引入与核心目标无关的复杂客户端逻辑
- 配置项保持少而明确，避免堆叠开关
- 命令名称简短，玩家与管理员职责分离
- 优先保证兼容性、可维护性和清晰度

## 代码约束

- 默认使用 Kotlin 编写主逻辑
- 仅在确有必要时引入 Mixin
- 注释保持精简，只解释不直观的逻辑
- 文件编码使用 UTF-8
- 构建目标为 Java 21

## 功能边界

- 玩家功能应尽量做到无需 OP
- 管理员功能统一通过配置文件和 `/sitadmin` 暴露
- 修改现有交互时必须考虑误触和方块原生行为冲突
- 任何新增功能都应优先考虑是否破坏“极简”目标

## 文档要求

- `README.md` 作为默认中文文档
- `README_EN.md` 提供英文版本
- 文档内容必须与当前实现保持一致
- 发布说明需要包含依赖、安装方式、命令、配置和构建要求

## 提交规范

- 按任务拆分提交
- 提交信息使用简洁的 Conventional Commit 风格
- 功能、文档、构建配置分别提交

## 验证要求

- 至少执行 `./gradlew build` 或 `gradlew.bat build`
- 如有 API 变动，优先依据本地映射源码修复
- 发版前确认产物命名和 GitHub Actions 工作流一致
